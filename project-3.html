Project 3

3# A Simple Website API project which shows the Current Weather .
• The aim of this project is to show the current weather of a particular Location .
• The user can enter the Location as input and get the weather information .
Used Languages and Frameworks.
• HTML and CSS was used for Building and styling of the website.
• JQuery Framework is used for the Logical part.
• Used API from https://openweathermap.org/ for getting the weather information.
